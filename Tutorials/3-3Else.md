# Else
> This function checks if no if statements are true
Write "elif"

```python
elif
```

Add a colon after "elif"

```python
elif:
```

Put any code in a new line

```python
elif:
  print("Hello")
```


## Examples

### Example 1:
```python
money = 1
if money == 1:
  print("You have",money,"money")
else:
  print("You dont have $1")
```
Run:
```
You have 1 money
```
